A propos

Poireaut

Un poireau moustachu, loupe en main. Clin d'oeil a Hercule Poirot et a la methode du detective qui relie patiemment les indices.

Collecteurs actifs
  • Pseudos — HTTP checker (50+ sites) + WhatsMyName (600+) + Maigret (3000+)
  • Emails — Gravatar, Holehe (120+ services), PGP (keys.openpgp.org), domaine jetable
  • Telephones — operateur, type, pays, fuseau (phonenumbers)
  • IP — geolocalisation, FAI, VPN/proxy (ip-api.com)
  • URLs — urlscan.io (IP/domaine/pays du serveur + screenshot)
  • Profils — enrichissement GitHub/GitLab/Gravatar/HackerNews, extraction de bio
  • GitHub — emails exposes dans les commits publics recents
  • Keybase — preuves cryptographiques cross-plateforme (Twitter, GitHub, Reddit...)
  • Wayback — anciennes versions archivees des profils (Internet Archive)
  • Avatars — hash perceptuel, correlation same_avatar_as
  • Domaines — WHOIS, DNS MX/TXT, sous-domaines via Certificate Transparency
  • Entreprises FR — registre officiel gouv.fr (SIREN, dirigeants, siege)
